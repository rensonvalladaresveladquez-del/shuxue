🔢 因数与倍数
✨ 什么是因数？
📌 如果 a × b = c (a,b,c都是整数)，那么 a 和 b 就是 c 的因数。
例如：2 × 3 = 6，所以2和3是6的因数。
6的因数还有：1, 2, 3, 6。
✨ 什么是倍数？
📌 如果 a × b = c，那么 c 就是 a 和 b 的倍数。
例如：3 × 4 = 12，所以12是3和4的倍数。
12的倍数有：12, 24, 36...
🌟 小秘诀：一个数最小的因数是1，最大的因数是它本身。
🌟 一个数最小的倍数是它本身，没有最大的倍数。
