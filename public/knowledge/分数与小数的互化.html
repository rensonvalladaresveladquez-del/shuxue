🔁 分数⇄小数互化

✨ 分数化小数： 用分子除以分母。

✨ 小数化分数： 一位小数写成十分之几，两位小数写成百分之几……再约分。

例：1/4 = 1÷4 = 0.25
0.75 = 75/100 = 3/4
🌰 常用互化：
1/2 = 0.5    1/4 = 0.25    3/4 = 0.75
1/5 = 0.2    2/5 = 0.4    3/5 = 0.6    4/5 = 0.8
1/8 = 0.125    3/8 = 0.375    5/8 = 0.625    7/8 = 0.875
🎮 游戏任务： 左边是分数或小数卡片，右边是它的互化目标。
将左边卡片拖到右边对应的目标框中，配对正确即可得分！
